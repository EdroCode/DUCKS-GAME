# GUIA COMPLETO DE COMANDOS - WORMS GAME

Para uma experência livre de problemas, é importante ler todas as NOTAS IMPORTANTES
para esclarecer certas nuances que possam ocorrer.

## MENU PRINCIPAL

Setas: Navegar entre opções
Enter: Selecionar opção
ESC: Voltar/Sair

---

## LEVEL SELECTOR

I: Importa mapa salvo
Setas: Navegar entre niveis
Enter: Jogar nivel selecionado
ESC: Voltar ao menu

---

## MODO PVP

MOVIMENTAÇÃO:
WASD/Setas: Movimento base (Norte/Sul/Est/Oeste)
Q: Noroeste
E: Nordeste
Z: Sudoeste
C: Sudeste

CONTROLES DE JOGO:
1: Trocar minhoca (próxima minhoca viva)
2: Trocar arma (ciclo: Jetpack → Escavadora → Bazuca → Mina → Dinamite → Nenhuma)
ESC: Voltar ao menu

---

## MAP CREATOR

NAVEGAÇÃO GERAL:
1: Alternar modo (Blocos → Objetos → Minhocas)
Seta Cima/Baixo: Navegar opções no menu lateral
Seta Esquerda/Direita: Navegar subopções
ESC: Voltar ao menu

MOUSE:
Clique Esquerdo: Adicionar elemento na posição
Clique Direito: Remover elemento (do modo selecionado) da posição

EDIÇÃO DO MAPA:
K: Adicionar linha abaixo
L: Adicionar coluna à direita
N: Remover última linha
M: Remover última coluna

EDIÇÃO DE MINHOCAS:
Enter: Ativar/desativar modo de edição de valores
0-9: Digitar valores numéricos (modo edição ativo)
Backspace ou Delete: Apagar último dígito
Seta Esquerda/Direita: Navegar entre atributos

SALVAR/CARREGAR:
E: Exportar estado atual "estado.txt"

---

## GAME OVER

ESC: Voltar ao menu principal

---

## HELP SCREEN

ESC ou Enter: Voltar ao menu principal

## NOTAS IMPORTANTES

Map Creator 

> Edição de números:
- Pressione Enter para ativar modo de edição
- Digite os números desejados (acumulam sequencialmente)
- Use Backspace/Delete para corrigir erros
- Os valores são salvos automaticamente

> Exportar estado:
- Após exportar um estado, um ficheiro com o nome "estado.txt" será salvo no diretório do jogo, este ficheiro não deve ser renomeado para evitar problemas
- Para salvar um mapa novo, o jogador deve salvar o conteudo do ficheiro em outro ficheiro e ir alterando o conteudo do "estado.txt" dependendo do estado que deseja importar no menu de seleção de níveis

PvP:
- O objetivo deste modo é simples, last one standing.
- As duas equipas devem jogar uma contra a outra, sendo responsabilidade das mesmas trocar as minhocas dependendo de como queiram organizar os turnos (ex: Assim tanto podem jogar 1 por 1 ou equipa por equipa)
- O jogo termina quando so restar minhocas de uma certa equipa

Level Selector:
- Use I para carregar estados salvos
- O jogo ira carregar o estado salvo no ficheiro "estado.txt" como um nivel novo
